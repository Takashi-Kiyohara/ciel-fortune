# Ciel Fortune — CLAUDE.md

## 事業の定義
「Ciel Fortune（シエル フォーチュン）」は、世界中を飛び回るCAが各国で出会った占い・タロット・神秘体験をキュレーションして届ける占いWebサービス。
裏側ではClaude APIを使うが、ユーザーには一切見せない。「CA × 世界の占い」という体験として提供する。

## 技術スタック
- Next.js (App Router) / TypeScript / Tailwind CSS
- Supabase (認証・DB)
- Claude API (占い文章生成 — 非公開)
- Stripe (決済)
- Vercel (ホスティング)

## 品質基準
1. AIであることをユーザーに見せない。全てCAキャラ「Misa」が語りかける体験にする
2. テクノロジー感のあるUI/文言を使わない（「生成しました」「分析結果」等NG）
3. 架空の情報を生成しない
4. 既存ファイルの上書き前は確認を取る

## セキュリティルール
- APIキーはコードに直書きしない（.envを使う）
- .gitignoreに.envを追加する
- rm / delete 等の削除コマンドは人間に確認してから実行
- pip install / npm install は承認後に実行
- 外部への送信は人間の承認後のみ

## デザイン方針
- カラー: ダークネイビー(#0d0d1a) × アンティークゴールド(#c4a265) × オフホワイト(#f8f5f0)
- テキスト: オフホワイト(#e8e4df) / ダークテキスト(#3d3630)
- フォント: 見出し=Shippori Mincho(明朝)、英語=Cormorant Garamond(セリフ)、本文=Noto Sans JP weight 300
- モチーフ: 空港・パスポート・搭乗券・星空
- アニメーション: 控えめで上品（Framer Motion使用）
- 原則: 余白で高級感を出す。装飾は引き算。border-radiusは4〜8px。グラデーション・影は極控えめ

## 禁止事項
- 「AI」「人工知能」「生成」「分析」等のテクノロジーワードをUIに表示すること
- 「Amazing」「Perfect」等の意味のない形容詞
- 機械的な箇条書きの占い結果（手紙風・語りかけ風にする）
